# code-gen
a code generator CLI tool

# how to install
> go get -u github.com/paulyung541/code-gen

# how to use
```bash
$ code-gen gen -d -n {your_project_name}
```
then the `{your_project_name}` project will created in your current directory like follow
```
./your_project_name
    main.go
    Makefile
```
